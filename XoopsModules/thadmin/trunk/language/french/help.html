Sommaire

  • Ce module permet de changer l'interface visuelle de votre panneau d'administration par l'utilisation de thémes spécifiques.

Installation

  • Pour installer le module, vous avez juste trois étapes :
    • Téléverser le module ThAdmin dans le répertoire des modules.
    • Installation du module dans l'interface d'administration de votre site
    • Au final, téléverser le fichier 'extras/cp_functions.php' à la place du fichier original dans 'include/cp_functions.php'. (Attention !! faites une sauvegarde du fichier original avant de l'écraser ou de le remplacer)
