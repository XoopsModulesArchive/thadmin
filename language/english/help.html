Summary

Install

  • For install the module, you have just three step :
    • Upload ThAdmin into the module directory.
    • Install module
    • Finally, upload the file 'extras/cp_functions.php' and replace the original xoops file in 'include/cp_functions.php'. (Warning!! please make a backup of the xoops files before perform these action)
